Will

Wiimote on linux

Will is a simple program made in python using python-cwiid and python-xlib to control the X server with a wiimote. The program is configurable to use the motion plus or the nunchuk as mouse pointer.

=== Use it ===

  1. install python-cwiid and python-xlib on your system
  2. edit config.yml with the following options:
  3. python will.py
  4. enjoy your wiimote

=== Configure it ===

The config.yml contains to parameter

  • mouse = [motion|nunchuck]
  • offset = [0-100]

mouse is the type of control you want to use, offset is the pixel offset at each movement

=== Info ===

I've created this software cause i'm far from an eletronic shop to buy ir leds and build a irda bar (yes, you can argue me that i didn't use 2 candles for trying, but i'm lazy) I'm very optimist about the motion plus, it's a great piece of hardware to use and to hack.

I've set for now simple actions with buttons:

  • A: left mouse click
  • B: right mouse click
  • +: zoom in
  • -: zoom out
  • home: reset pointer to center of screen

I'll get them configurable very soon.

Best use is with a raspberry pi, on board screen and a television, you can browse internet directly from wiimote without a mouse and a keyboard. I'll post soon some videos.
