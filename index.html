=== Wiimote on linux ===

Will is a simple program made in python using python-cwiid and python-uinput ( https://github.com/tuomasjjrasanen/python-uinput ) The program is configurable to use the motion plus or the nunchuk as mouse pointer.

Will is a spare time project, so if you want to support me with a little donation, it would be very appreciated

=== Use it ===

  1. install python-cwiid and python-uinput on your system
  2. edit config.yml
  3. python will.py
  4. set permission for udev on your wiimote
  5. enjoy your wiimote

=== Configure it ===

The config.yml contains to parameter

  • mouse = [motion|nunchuck]
  • offset = [0-100]
  • a = [keyboard key for key a]
  • b = [keyboard key for key b]
  • up = [keyboard key for key up]
  • down = [keyboard key for key down]
  • left = [keyboard key for key left]
  • right = [keyboard key for key right]
  • plus = [keyboard key for key plus]
  • minus = [keyboard key for key minus]

mouse is the type of control you want to use, offset is the pixel offset at each movement

=== Set permission for udev ===

First load the kernel module uinput and assure that you have it in the kernel:

**** modprobe uinput

then you can put uinput device permission to /dev/uinput

**** chmod 0666 /dev/uinput
